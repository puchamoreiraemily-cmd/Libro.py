# Tarea de POO - Semana 2

Hola. En este repositorio subí mi tarea de la Semana 2. Creé un programa en Python que representa un objeto real (un libro) usando Programación Orientada a Objetos.

### Lo que incluye el código:
* **Clase:** Libro
* **Atributos:** Título, autor y número de páginas.
* **Métodos:** Uno para mostrar los datos en pantalla y otro para simular que estás leyendo.
* **Objetos:** Creé dos libros diferentes para probar que todo funciona bien.

Desarrollado en Visual Studio Code con Python
